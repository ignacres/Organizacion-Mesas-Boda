Cómo obtener tu API Key
Solo necesitas este paso — el documento ya está configurado

1. Abre Google Cloud Console
Ve a console.cloud.google.com e inicia sesión con tu cuenta de Google.

2. Crea o selecciona un proyecto
Arriba a la izquierda, haz clic en el selector de proyectos → Nuevo proyecto → ponle un nombre (ej: "Boda") → Crear.

3. Activa la Google Sheets API
Ve a APIs y servicios → Biblioteca → busca "Google Sheets API" → haz clic → Habilitar.

4. Crea la clave de API
Ve a APIs y servicios → Credenciales → Crear credencial → Clave de API.
En las restricciones, selecciona Google Sheets API en el desplegable y guarda.
Copia la clave generada y pégala en el campo de la app.

5. Asegúrate de que el documento es público
En tu Google Sheet → Compartir → Cualquier persona con el enlace → rol Lector.
